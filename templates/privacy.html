Privacy Policy

Last Updated: February 16, 2026

Welcome to BookGenie. Hum aapki privacy ka poora dhyan rakhte hain. Ye policy batati hai ki hum aapka data kaise collect aur secure karte hain.

Quick Summary: Hum aapka data bechte nahi hain. Hum sirf aapko behtar book recommendations aur chatbot services dene ke liye data process karte hain.

1. Information We Collect

Hum niche di gayi jankari collect karte hain:

  • Catalog Data: Jo books aap hamare store par dekhte hain, wo hamare MongoDB database se aati hain.
  • Chat Conversations: Chatbot ke saath hone wali baatcheet OpenAI APIs ke dwara process ki jati hai taaki aapko AI-based jawab mil sakein.

2. How We Use Data

Aapka data in kaamo ke liye use hota hai:

  • Website par books ko filter aur search karne ke liye.
  • AI Chatbot ke zariye book prices aur recommendations dene ke liye.
  • Database performance ko optimize karne ke liye.

3. Third-Party Sharing

Hum aapka data sirf in trusted services ke saath share karte hain:

  • OpenAI: Sirf chatbot messages ko reply generate karne ke liye bheja jata hai.
  • MongoDB: Aapka local library data yahan safely store hota hai.

4. Security

Humne aapki security ke liye CORS protection lagayi hai. Saath hi, saari sensitive API keys hamare server par .env files mein secure hain.
